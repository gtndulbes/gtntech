Sistem Monitoring & Kontrol Kandang Ulat Hongkong

Berbasis IoT dengan Fuzzy-PID dan Self-Monitoring System

ESP32-C3 SHT31-D MQ-135 Fuzzy-PID Self Monitoring
Latar Belakang

Budidaya ulat Hongkong (Tenebrio molitor) skala UMKM masih menghadapi kendala mortalitas larva tinggi (20–40%) akibat ketidakstabilan suhu, kelembaban, dan kualitas udara. Sistem ini dirancang untuk menggantikan pemantauan manual dengan kontrol otomatis berbasis IoT dan algoritma cerdas.

Proyek Tugas Akhir S-1 Teknik Elektro
Tujuan
  • Memantau dan mengontrol lingkungan kandang secara real-time
  • Menerapkan kontrol Fuzzy-PID untuk stabilitas adaptif
  • Mendeteksi dini kegagalan perangkat keras (self-monitoring)
  • Menurunkan mortalitas larva hingga ≥15%
  • Menyediakan dashboard akses jarak jauh melalui web
